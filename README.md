# TypeScript-Search-Ranker

![CI](https://github.com/skylerblue333/TypeScript-Search-Ranker/workflows/CI/badge.svg)

Production-ready TypeScript Express API for ranker operations.

## Features
- Fully typed Express handlers
- Supertest integration testing
- Docker containerization
- GitHub Actions CI Pipeline

## Quick Start
```bash
npm ci
npm test
npm run build && npm start
```
